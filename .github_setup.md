# GitHub Repository Setup Guide

## Step-by-Step Instructions

### 1. Create Repository on GitHub

1. Go to https://github.com/new
2. Fill in:
   - **Repository name**: `hfl-multitree`
   - **Description**: `Hierarchical Federated Learning with MultiTree All-Reduce - CS 8125 Project`
   - **Visibility**: Public (recommended for portfolio) or Private
   - **DO NOT** check "Initialize with README"
3. Click "Create repository"

### 2. Push Your Code
```bash
# From your project directory
cd ~/Cloud\ Computing/hfl_multitree_project

# Commit all files
git commit -m "Initial commit: Complete HFL+MultiTree implementation

- Implemented MultiTree all-reduce with O(log n) complexity
- Integrated AHFLP adaptive aggregation optimization
- Achieved 80% reduction in training time and energy
- Comprehensive simulation framework with SimPy
- Complete documentation and visualizations"

# Set main branch
git branch -M main

# Add remote (replace Kazi-Nasif with your GitHub username)
git remote add origin https://github.com/Kazi-Nasif/hfl-multitree.git

# Push to GitHub
git push -u origin main
```

### 3. Enhance Your Repository

#### Add Topics/Tags
In your GitHub repository page:
- Click on the gear icon next to "About"
- Add topics: `federated-learning`, `distributed-systems`, `pytorch`, `optimization`, `cloud-computing`, `machine-learning`, `simulation`

#### Create Releases
```bash
# Tag your current version
git tag -a v1.0.0 -m "Release v1.0.0: Complete implementation with 80% improvements"
git push origin v1.0.0
```

#### Add Images to README
1. Upload your plots to GitHub:
   - Create `results/plots/` in your repo
   - GitHub will host the images
   
2. Or use direct paths in README:
```markdown
   ![Performance](results/plots/performance_comparison.png)
```

### 5. Add GitHub Actions (Optional)

Create `.github/workflows/tests.yml`:
```yaml
name: Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v3
    - name: Set up Python
      uses: actions/setup-python@v4
      with:
        python-version: '3.10'
    - name: Install dependencies
      run: |
        pip install -r requirements.txt
    - name: Run tests
      run: |
        python test_multitree.py
```

## Useful Git Commands
```bash
# Check status
git status

# View commit history
git log --oneline

# Create new branch for features
git checkout -b feature/new-topology

# Undo last commit (keep changes)
git reset --soft HEAD~1

# Update from remote
git pull origin main

# View remote URL
git remote -v
```

## Troubleshooting

**Problem**: Large files rejected
```bash
# Use Git LFS for large files
git lfs install
git lfs track "*.pth"
git lfs track "*.pt"
git add .gitattributes
```

**Problem**: Wrong remote URL
```bash
git remote set-url origin https://github.com/Kazi-Nasif/hfl-multitree.git
```

**Problem**: Need to change commit message
```bash
git commit --amend -m "New message"
git push --force
```
